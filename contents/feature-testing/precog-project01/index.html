Precog has been acquired by RichRelevance!

Two months after launching from beta, Precog is delighted to announce it has been acquired by RichRelevance!

As a part of RichRelevance, Precog will be working to power the next generation of advanced analytical applications for the retail industry.

RichRelevance is passionate about contributing back to the community, and has decided to open source the entire Precog code base, which can now be found on Github.
